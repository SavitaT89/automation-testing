Mobile Automation Framework – WizzAir App

Tech Stack
- Java
- Appium
- Cucumber BDD
- Maven

Supported Platforms
- iOS

How to Run Tests
1. Install Appium
2. Start Appium server
3. Connect device
4. Update Device details
5. Run:

mvn clean test

Framework Design
- Page Object Model
- DriverManager pattern
- Explicit waits (no hard sleeps)
- Cross-platform locators
